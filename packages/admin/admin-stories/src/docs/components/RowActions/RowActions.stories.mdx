# RowActions (Icon Buttons & More-Menu)

A list of IconButtons with a "More" icon to open a menu of additional ListItems. <br/>
Generally, RowActions are used within table rows or similar lists of items.

## Icon Actions

IconButtons can be added using the `iconActions` prop. <br/>
The value should be an array of `IconButton` props objects.

## Menu Actions

Actions can be added to the "More" menu using the `menuActions` prop. <br/>
The value should be an array of `ListItem` props objects.

<Canvas>
    <Story id="stories-components-rowactions--menuactions-with-props-object" />
</Canvas>

### Render custom menu actions

You can define a node instead of the props object for a `Divider` or other use cases where the default `ListItem` is not an option.

_Note: Closing the menu must be handled manually in this case by using the `closeMenu` function, available when passing a function that returns the node._

<Canvas>
    <Story id="stories-components-rowactions--menuactions-with-react-node" />
</Canvas>

### Manually handle closing the menu

By default, the "More" menu is closed when clicking the item. <br/>
This can be prevented by setting `preventCloseOnClick` in the props object or by rendering a custom node.

The `closeMenu` function is passed as a parameter to the `onClick` function of the props object. <br/>
When rendering a custom node, `closeMenu` is available when passing a function that returns the node.
